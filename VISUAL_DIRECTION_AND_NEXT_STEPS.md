# A2F Visual Direction And Next Steps

## Updated Creative Direction

The new A2F site should not inherit the current site literally. It should inherit the brand, but modernize the presentation.

### What Stays

- A2F red as the primary brand accent
- strong typography roots from the current brand
- performance-focused, high-contrast gym attitude
- confidence, structure, and credibility

### What Changes

- less dependence on charcoal for every section background
- cleaner spacing and more premium composition
- more light, depth, and contrast variation
- more modern landing-page pacing
- stronger mobile-first structure
- more intentional SEO architecture for two locations

## Visual Design Principles

### 1. Modern Gym, Not Old-School Bodybuilding Template

The site should feel:

- sleek
- sharp
- premium
- visually controlled
- bold without being cluttered

Avoid:

- every section on black
- cramped layouts
- generic boxed WordPress-style content blocks
- noisy overlays and overuse of red

### 2. Dynamic Lighting

Use lighting direction in the design system:

- strong dark-to-light hero imagery
- subtle glows or edge-light effects around key image areas
- red highlights used as accents, not flood color
- controlled contrast that makes the brand feel elevated

### 3. Cleaner Color Use

Primary palette should still come from A2F, but the ratio changes:

- red for CTA, accents, and key emphasis
- off-white / white sections for breathing room
- charcoal only where it increases drama or contrast
- light grays and soft neutral surfaces for modern polish

### 4. Strong UX Hierarchy

Users should immediately understand:

- there are two experiences
- which one is right for them
- what the next step is

## Recommended Visual System

### Color Ratio

- 10% red accents
- 25% dark backgrounds / dramatic surfaces
- 65% white / off-white / neutral surfaces

### Typography Direction

Current site uses:

- `Open Sans`
- `Oswald`

Recommendation:

- keep `Oswald` for brand-impact headings
- use a cleaner modern sans for body and interface if needed during design exploration
- or retain `Open Sans` initially and refine later

Best approach:

- primary headings: bold condensed display style
- body copy: clean modern sans with strong readability

### Layout Style

- large image-led sections
- asymmetrical but controlled composition
- strong spacing rhythm
- high-value CTAs with breathing room
- sections that alternate between dark drama and bright clarity

## SEO Structure Requirements

This site needs to rank and index well for both locations, so structure matters early.

### Must-Haves

- one page for each location intent
- clean URL structure
- page-specific metadata
- location-specific H1s and supporting copy
- internal linking between location pages and homepage
- FAQ content for long-tail search support
