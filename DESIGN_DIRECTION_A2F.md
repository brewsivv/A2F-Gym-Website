# A2F Design Direction

## Overall Direction

The references point toward a visual system that is:

- modern
- aggressive but clean
- image-led
- high-contrast
- editorial in composition
- premium rather than gritty

For A2F, we should adapt this energy to the existing brand instead of copying the exact neon-green look.

## What To Borrow From The References

### 1. Bold Hero Composition

Use:

- oversized typography
- strong crop-focused photography
- layered image panels
- floating UI-like stat cards
- a premium landing-page feel instead of a standard gym brochure layout

### 2. Dynamic Lighting

Use:

- dramatic subject lighting
- controlled glow effects
- bright accent lighting around key images
- dark-to-light transitions instead of flat dark blocks

### 3. Strong Card-Based Layouts

Use:

- modular feature cards
- high-end membership cards
- stacked content panels
- asymmetrical section layouts that still feel organized

### 4. Minimal but Bold Typography

Use:

- large uppercase headlines
- short supporting paragraphs
- restrained copy blocks
- oversized section transitions or background words

### 5. Premium Motion Direction

Later in build:

- staggered reveals
- image parallax or subtle depth
- hover elevation on cards
- CTA interactions with crisp contrast shifts

## What To Avoid Copying Literally

- neon lime as the main accent color
- all-black sections from top to bottom
- overly busy abstract overlays
- layouts that sacrifice readability for style

## A2F Brand Translation

## Color Direction

Instead of neon green, use A2F red as the signature accent.

### Primary Palette

- Brand red: `#ED1C23`
- Accent red hover: `#D91E25`
- Soft off-white: `#F5F5F2`
- Clean white: `#FFFFFF`
- Deep ink: `#111111`
- Charcoal support: `#222222`
- Neutral gray surfaces: `#E9E9E6`

### Color Strategy

- use red for action, emphasis, highlights, and energy lines
- use black and ink for drama in hero and key conversion sections
- use white and soft neutral sections to make the site feel premium and breathable
- use charcoal sparingly as a support tone, not the default canvas

## Typography Direction

Current brand foundation:

- `Oswald`
- `Open Sans`

Recommended use:

- keep `Oswald` for impact headlines
- use a cleaner contemporary sans for body and interface if needed during build exploration

If staying close to current brand:

- Headings: `Oswald`
- Body/UI: `Open Sans`

If slightly modernizing:

- Headings: `Oswald`
- Body/UI: `Manrope`, `Plus Jakarta Sans`, or `Inter`

Best recommendation:

- preserve `Oswald`
- modernize body copy if it improves polish and readability

## Page Personality

### Homepage

Should feel:

- cinematic
- premium
- fast
- decisive

Visual traits:

- large hero with layered photo treatment
- two-path split with polished cards
- alternating light and dark sections
- stat chips, callout cards, and clean CTA blocks

### Private Training Path

Should feel:

- premium
- focused
- guided
- transformation-led

Visual traits:

- stronger editorial layout
- more intimate imagery
- deeper contrast
- stronger emphasis on people, coaching, and outcomes

### Public Gym Path

Should feel:

- structured
- energetic
- elevated
- membership-driven

Visual traits:

- cleaner grid system
- stronger feature card layouts
- more environment and equipment emphasis
- membership pricing cards that feel modern and sharp

## Section-Level Styling Notes

## Hero

Inspiration taken from:

- bold wordmark-scale typography
- dramatic athlete crop
- floating CTA / content cards

For A2F:

- use one dominant athlete/gym image
- overlay large clean headline
- include two CTA paths
- add one or two floating stat/value cards
- use red accent glow or edge highlight instead of neon green

## Two-Path Split Section

Inspiration taken from:

- modular cards
- asymmetrical editorial structure

For A2F:

- create two high-end destination cards
- make each card feel visually distinct
- private side slightly more refined and coached
- public side slightly more performance and access focused

## Membership Section

Inspiration taken from:

- sharp pricing blocks
- bold contrast cards

For A2F:

- one card can invert color for emphasis
- use “best value” treatment on paid-in-full
- keep plan comparison easy to scan on mobile

## Results / Testimonials

Inspiration taken from:

- strong portrait-driven panels
- minimal copy framing

For A2F:

- use real faces and outcomes
- combine testimonial quote with concise supporting context
- avoid generic carousel-only treatment if possible

## Businesses At A2F

Inspiration taken from:

- modular showcase blocks
- brand ecosystem feeling

For A2F:

- present businesses like premium amenities within the larger gym experience
- use a clean card or tile system

## SEO And UX Implications

These references are visually bold, but we still need strong SEO and usability.

### Requirements

- real text over images, not text embedded into graphics
- fast-loading optimized assets
- semantic heading structure
- clean mobile stacking
- clear primary CTA per section

### Important Balance

The site can look premium and still be indexable if:

- typography remains HTML text
- animation is restrained
- layout complexity does not harm mobile scanning

The best A2F version of these references is:

- less neon
- less template-like
- more brand-authentic
- more breathable
- more premium
- still visually powerful and unmistakably fitness-focused
