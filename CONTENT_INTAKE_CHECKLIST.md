# A2F Content Intake Checklist

## Purpose

Use this checklist to gather the remaining client content in the right order so design and development can move without bottlenecks.

## Highest-Priority Decisions

### 1. Final Naming

Need confirmation on:

- exact public-facing name for the existing location
- exact public-facing name for the new location

Recommended direction:

- `A2F Private Gym`
- `A2F Public Gym`

Alternative if they want more polish:

- `A2F Training`
- `A2F Gym`

### 2. Contact Routing

Need to know:

- which email should receive private training inquiries
- which email should receive public gym membership inquiries
- which email should receive corporate discount inquiries
- which email should receive business-unit or leasing inquiries

### 3. Membership Signup Workflow

Need confirmation on:

- whether users can join online immediately
- whether signup happens through a third-party system
- whether day passes can be purchased online
- whether tours are scheduled manually or through software

## Brand Assets

### Required

- logo files
- alternate logo versions if available
- favicon
- any sub-brand lockups if they want separate naming treatments

### Helpful

- brand guide if one exists
- old flyers or promo graphics
- social media graphics they like

## Photography and Media

### Private Gym

- exterior photos
- interior floor photos
- training photos
- coach photos
- client transformation photos if approved

### Public Gym

- exterior photos or renderings
- interior photos or renderings
- equipment photos
- access / entry visuals
- environment / community shots

### Businesses at A2F

- business logos
- owner photos if available
- service visuals

## Copy Inputs by Page

## Home

Need:

- preferred homepage headline direction
- short master brand description
- short blurbs for private and public paths
- strongest brand differentiators

## Locations

Need:

- both addresses
- both hours
- parking instructions
- any notes about appointments, staffed hours, or access windows

## Private Training

Need:

- updated service descriptions
- coaching philosophy
- trainer bios
- private consultation process
- any pricing they want public

## Public Gym

Need:

- final approved hero copy
- any additional amenities
- equipment highlights
- access details
- staffed support details if any

## Memberships

Need:

- confirmation of all pricing
- setup fee details
- contract terms
- freeze policy
- cancellation policy
- guest policy
- corporate discount details

## Businesses at A2F

Need for each business:

- name
- service description
- why it benefits members
- logo
- contact info
- booking link if any
- launch timing

## About

Need:

- founder story
- why A2F was created
- why the second location was opened
- mission statement if they have one

## Results / Testimonials

Need:

- best private gym testimonials
- public gym testimonials when available
- before-and-after permissions
- names or initials preference

## Contact / Tour / Join

Need:

- main phone number
- general email
- form routing preferences
- tour scheduling process

## Waiver

Need:

- current waiver link or file
- whether private and public use the same waiver
- any minors policy

## FAQ

Need:

- top repetitive questions from staff
- top repetitive questions from members
- objections they hear most before sign-up

## Recommended Collection Order

### Phase 1

- naming
- addresses
- hours
- pricing confirmation
- membership workflow

### Phase 2

- page copy
- testimonials
- bios
- FAQs

### Phase 3

- references
- photography
- business-unit assets

## Best Way To Send Content

Most efficient format:

- send one page at a time
- include rough copy even if not polished
- include links for references
- include notes like `must keep`, `optional`, or `rewrite this`
